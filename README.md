## 项目简介

基于 Vite + React 的前端小工具项目，支持本地开发与 GitHub Pages 自动部署。

## 功能概览

- 本地开发热更新
- 生产构建与预览
- GitHub Actions 自动部署到 Pages

## 使用手册

### 快速开始

1. 点击右上角“导入”，选择已有数据包（`.zip`）即可加载。
2. 也可以将 `.zip` 直接拖拽到页面中间区域完成导入。
3. 若是空项目，可直接开始新建指令与上传图片。

### 新建指令与参考图

1. 点击“添加指令”新增一条指令条目。
2. 在“指令内容”中填写编辑指令。
3. 在“参考素材”区域拖拽或点击“添加”上传参考图。

### 目标结果与模型名称

1. 在“目标结果”区域上传目标图（可拖拽或点击添加）。
2. 为每个目标结果填写对应“模型名称”。
3. 如果从数据包导入且模型名称缺失，系统会提示补充。

### 导出数据包

1. 确保所有目标结果都填写了模型名称。
2. 点击右上角“导出数据包”，会生成并下载 `.zip`。
3. 导出的结构包含每个 case 的图片与 `metadata.json`。

### 结构预览

1. 点击右上角“结构预览”可查看当前数据结构的 JSON 预览。
2. 该预览不会写入文件，仅用于检查结构是否符合预期。

## 本地运行

**前置条件：** 已安装 Node.js（建议 18+）

1. 安装依赖：`npm install`
2. 启动开发：`npm run dev`
3. 打开浏览器访问：`http://localhost:3000`

## 构建与预览

- 构建：`npm run build`
- 本地预览构建产物：`npm run preview`
